权限说明

SSH 会读取 Docker 状态与模板文件，并在同步时写入模板、创建备份、刷新目标容器。

本地图标库

默认使用 HD Icons；后续可以继续添加更多图标库一起检索。

默认图标库可以直接下载。点击每个图标库右侧的铅笔可修改名称和下载链接；本地图标会复制到下方映射目录并写入 Unraid 本地路径。

选择本地图标后会写入这个宿主机路径，需和 Docker 的图标目录映射对应。
